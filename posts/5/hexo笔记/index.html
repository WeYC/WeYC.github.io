Hexo-Themes-Next主题：

  • 配置：

    1. cd 到 Blog目录下的 themes，执行 git clone https://github.com/iissnan/hexo-theme-next themes/next 下载主题文件

    2. 修改_config.yml里的themes字段 theme: next

    3. 验证主题一次输入以下命令：

​ hexo clean清除已经生成的静态文件

​ hexo g 编译生成静态文件

​ hexo s部署到本地服务器，在浏览器输入http://localhost:4000打开页面

​ hexo d上传到Github
